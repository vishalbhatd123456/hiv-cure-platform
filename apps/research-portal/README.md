# Research Portal

React / Next.js application for researchers.

Primary workflows:

- Cohort exploration
- Sample tracking
- Simulation requests
- AI run review
- Research reports
- Workflow status visibility

## Public Research Radar

The first frontend slice is a static public read-only view at:

```text
apps/research-portal/public-research
```

It shows the latest 100 HIV cure-related research records from Europe PMC,
sorted by publication date.
